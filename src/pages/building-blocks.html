Making a powerful tool usable for beginners

Overview

Foundation, one of the most popular front-end coding frameworks, needed a way to grow their audience beyond die-hard fans and experts to include a new generation of coders.

Role

Primary Designer

Challenge

Satisfying Beginners and Experts

As Foundation grew, only a fraction of users were using its full power because of a steep learning curve. I needed to make the features more accessible without diminishing their power, since the product at its core aims to be the most advanced front-end framework available.

Research

It was critical to understand when and why users decide to learn more about and adopt the framework into their workflows. Conversation and analysis of online questions revealed key insights.

  • User Support Log Analysis
  • User Interviews
  • Product Audit
Strategic Components

The design direction centered on finding a way to thoughtfully suggest key code snippets. Mapping component types Foundation provides revealed six main categories to build code snippets off of - navigation, status, containers, forms, controls, and media.

The Result

Building Blocks jumpstart Any Project

A community-driven platform of quick, plug-and-play components encourages engagement from beginner to expert by providing solutions that fit any project without hassle.

Plug And Play Components

Large, image-driven cards help coders find the right ones, verify compatibility and socially assess quality. Highlighting authors encourage submissions. They can then download it on the next page.

Learn As You Browse

Toggling aspects of the underlying code helps learners focus on how others used foundation successfully and efficiently.

Kits For Common Projects

Bundling common components into kits of the most common project types lowers the intimidation factor of starting your first project.

Encouraging Exploration

A “related components” section and “search for more” in kits helps users discover more than they initially came for and find missing components for unique projects.

Feedback Foundation Building Blocks launched, with users reporting experiencing a 10x decrease in development time, 2x higher signups in advanced Foundation courses, and contributing 20% of the building blocks in the first week.
